Requirements

Your server require these features:

  • PHP >= 7.2.5
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

These are the requirements for Laravel project.

Installation

 

  • Create a database and add User to the database.
  • Upload the database.sql file to your database from PHPMyadmin.
  • Upload the SupportDesk.zip to your home directory.
  • Extract the SupportDesk.zip file.
  • Open SupportDesk Application folder and edit the .env file.

 

  • You need to add the database details and app URL on this file.
  • Please note that the .htaccess file is present at the new directory.

Control Panel

You can access control panel from http://YourDomain.com/Directory/Home

if you have installed software in support directory then you can access control panel from

http://www.YourDomain.com/support/home

Login Information

Email: info@devzsudio.com

Password: demopassword

You can change these details after login.

Tickets

You can manage open tickets after login.

Manage tickets URL: http://www.YourDomain.com/Directory/Home

The open tickets will be shown on the dashboard. Admin can view all the tickets. And for other users (Staff) it depends on the permission set by the administrator. You can view closed tickets on Tickets section.

Settings

You can access settings on menu after login.

1. Departments: You can add all the departments that you need on support ticket system. Please note that the departments need to be separated by comma.

2. From email: From this email, the ticket updates are sent to client and staff.

3. Slack webhook URL: If you want to receive notifications via slack, enter slack webhook URL. You can generate it from https://my.slack.com/services/new/incoming-webhook/. This one is optional.

Staffs

From staff page, you can add new staff, edit staff.

If you want to add new staff go to Staffs -> New Staff. You need to add name, email, password, and department. You can choose multiple departments. Press control and select departments.

Customize

You can customize theme or style from SupportDesk/resources/views

You can view template files in that directory.

If you need to edit the CSS or JS, you can do this from Public directory.
